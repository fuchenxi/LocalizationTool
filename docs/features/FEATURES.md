# 功能特性总览

## 🎯 核心功能

### 1. 📱 项目信息展示
- ✅ 自动查找并解析 `Info.plist` 文件
- ✅ 显示 App 图标（100x100，圆角设计）
- ✅ 显示 App 名称
- ✅ 显示版本号
- ✅ 显示 Bundle ID
- ✅ 显示 Info.plist 相对路径

### 2. 🔍 查重去重
- ✅ 扫描所有语言文件夹（`.lproj`）
- ✅ 可配置忽略文件夹（`Pods|DerivedData|build` 等）
- ✅ 按语言分 Tab 显示重复项
- ✅ 表格显示详细信息：
  - Key - 重复的键名
  - Value - 对应的值
  - 行号 - 文件中的行号
  - 出现次数 - 重复次数
  - **操作 - 打开文件按钮** ⭐
- ✅ 删除重复项，保留最后一个
- ✅ **保留所有注释和空行**
- ✅ Git diff 只显示删除的行

### 3. 📥 导入多语言
- ✅ 支持 zip 包导入
- ✅ 自动识别语言文件
- ✅ 匹配项目中的 `.lproj` 文件夹
- ✅ 追加到文件末尾
- ✅ 用版本号注释包裹新增内容
- ✅ 支持多种 zip 格式

## ⭐ 交互特性

### 双击复制
在扫描结果表格中，**双击以下列可复制内容**：
- 📋 **Key 列** → 复制键名
- 📋 **Value 列** → 复制值
- 📋 **行号列** → 复制行号
- 📋 **出现次数列** → 复制次数

**使用场景**：
- 复制 Key 去全局搜索
- 复制 Value 去对比翻译
- 复制行号去手动查看

**反馈**：
- 自动复制到剪贴板
- 左侧日志显示：`📋 已复制 Key: action_cancel`

---

### 点击打开文件

在扫描结果表格中，**点击"📂 打开"按钮**：
- 📂 在编辑器中打开对应文件
- 🎯 自动跳转到指定行号
- 👆 鼠标悬停显示手型光标
- 💫 点击时背景闪烁（视觉反馈）

**支持的编辑器**（按优先级）：
1. **Xcode** - `xed --line <line> <file>`
2. **VSCode** - `code -g <file>:<line>`
3. **Sublime Text** - `subl <file>:<line>`
4. **Atom** - `atom <file>:<line>`
5. **系统默认** - `open <file>`

**工作流程**：
```
点击"打开" 
    ↓
日志显示: 📂 正在打开 en.lproj/Localizable.strings 第 12 行...
    ↓
Xcode 打开文件并跳转到第 12 行
    ↓
查看上下文，决定保留哪个值
```

## 🛡️ 安全特性

### 1. 保留文件结构
删除重复项时：
- ✅ 保留所有 `//` 注释
- ✅ 保留所有 `/* */` 注释
- ✅ 保留所有空行
- ✅ 保留版本号注释
- ✅ 只删除重复的 key-value 行

### 2. Git 友好
- ✅ 最小化文件变更
- ✅ diff 清晰易读
- ✅ 只显示真正删除的行
- ✅ 易于 Code Review

### 3. 二次确认
所有危险操作都有确认对话框：
- ⚠️ 删除重复项前确认
- ⚠️ 导入多语言前确认
- ⚠️ 显示影响范围和数量
