# 导入多语言功能指南

## 🎯 新设计的导入界面

### 主要改进
- ✅ **文件夹路径选择** - 可以选择任意包含 ZIP 的文件夹
- ✅ **ZIP 文件列表** - 自动列出所有 ZIP 文件
- ✅ **按时间排序** - 最新的文件在最上面
- ✅ **默认选中** - 自动选中最新的文件
- ✅ **文件信息** - 显示文件名、大小、修改时间

## 🖱️ 使用方法

### 1. 默认流程（最简单）
```
1. 选择项目路径
   ↓
2. 切换到"导入多语言" Tab
   ↓
3. 自动加载 ~/Downloads 下的所有 ZIP 文件
   ↓
4. 最新的文件已被选中（蓝色高亮）
   ↓
5. 点击"开始导入多语言"
   ↓
6. 确认并导入 ✅
```

### 2. 更改文件夹
```
1. 点击"更改"按钮
   ↓
2. 选择其他文件夹（如：桌面、文档等）
   ↓
3. 自动刷新列表
   ↓
4. 显示新文件夹中的所有 ZIP 文件
```

### 3. 刷新列表
```
1. 下载了新的 ZIP 文件
   ↓
2. 点击"刷新"按钮
   ↓
3. 重新扫描当前文件夹
   ↓
4. 自动选中最新的文件
```

### 4. 选择其他文件
```
1. 在列表中点击其他 ZIP 文件
   ↓
2. 该文件变为蓝色高亮
   ↓
3. 点击"开始导入多语言"
   ↓
4. 导入选中的文件
```

## 📊 列表显示格式

### 文件信息
```
文件名  |  大小  |  修改时间
stringsdownload_20251029181000.zip  |  2.35 MB  |  2025-10-29 18:10:00
```

### 排序规则
- 按修改时间**倒序**排列
- 最新的文件在**最上面**
- 方便快速找到最近下载的文件

### 颜色标记
```
第一个文件（最新）: 蓝色文字（提示这是最新的）
其他文件: 黑色文字
选中的文件: 蓝色高亮背景 + 白色文字
悬停的文件: 灰色背景
```

## 🎯 自动化特性

### 1. 自动扫描
选择项目后，自动扫描 Downloads 文件夹中的所有 ZIP 文件。

### 2. 自动排序
按修改时间倒序排列，最新的在前。

### 3. 自动选中
默认选中第一个（最新的）文件，无需手动点击。

### 4. 智能状态
- 有文件：显示"共 X 个文件"，启用导入按钮
- 无文件：显示"未找到 ZIP 文件"，禁用导入按钮

## 📝 注意事项

### 文件名格式
支持任何以 `.zip` 结尾的文件：
- ✅ `stringsdownload_20251029181000.zip`
- ✅ `localization.zip`
- ✅ `translations_v1.2.zip`
- ✅ 任何名称.zip

### 文件大小
- 显示为 MB（兆字节）
- 小文件（< 0.01 MB）显示为 0.00 MB
- 大文件正常显示

### 时间格式
- 使用本地时区
- 格式：`年-月-日 时:分:秒`
- 便于识别和对比
