# 配置缓存功能说明

## 🎯 功能说明

工具会自动记住您上次的使用状态，下次打开时自动恢复。

---

## 💾 缓存的内容

### 1. 项目路径
```
上次选择的项目路径会被保存
下次启动时自动加载
无需重新选择
```

### 2. 导入文件夹
```
上次使用的 ZIP 文件夹路径
下次导入时默认使用
（默认: ~/Downloads）
```

---

## 📁 配置文件位置

```
~/.ios_localization_tool.json
```

### 文件格式
```json
{
  "last_project_path": "/Users/xxx/Desktop/ChillFit",
  "last_import_folder": "/Users/xxx/Downloads"
}
```

---

## 🔄 工作流程

### 首次使用
```
1. 启动工具
2. 选择项目路径
3. 路径自动保存到配置文件
```

### 再次使用
```
1. 启动工具
2. 自动加载上次的项目路径
3. 自动显示项目信息
4. 直接开始使用功能 ✅
```

```
点击"浏览"
    ↓
文件选择对话框默认打开上次的路径

选择新项目
    ↓
新路径自动保存
```

---

## 🔧 配置管理

### 自动验证
```
读取配置时会验证路径是否存在
如果不存在（文件被删除/移动）
则不加载，提示选择新路径
```

### 自动更新
```
每次选择新路径时自动保存
每次更改导入文件夹时自动保存
无需手动操作
```
